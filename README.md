# SkyrimXI Music Pack

**The Elder Scrolls V: Skyrim OST as an XIPivot overlay for Final Fantasy XI**

Replaces select FFXI zone, town, dungeon, and battle music with Jeremy Soule's iconic Skyrim soundtrack — mood-matched so dark zones get dark tracks, peaceful towns get warm tracks, and battles get epic orchestral combat music.

Boss themes, cutscenes, and title screens are intentionally left as original FFXI music.

---

## Installation (XIPivot)

1. Download the latest release
2. Extract into your XIPivot overlays directory:
   - **Windower:** `windower/addons/xipivot/overlays/SkyrimXI/`
   - **Ashita:** `ashita/addons/xipivot/overlays/SkyrimXI/`
3. Enable the `SkyrimXI` overlay in XIPivot
4. Enjoy Skyrim music in Vana'diel
