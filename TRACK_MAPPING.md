- **Nation mission-specific cutscenes** — Too context-dependent

## CONVERSION WORKFLOW (Per Track)

1. Download audio from YouTube → MP3
2. Convert MP3 → WAV (16-bit, 44.1kHz) in Audacity
3. Normalize loudness to ~10 LUFS (match FFXI levels)
4. Find loop point sample number in Audacity
5. Calculate hex: sample ÷ 16 → round → convert to hex
6. Run through XiMC: WAV → BGW with loop point
7. Rename output to correct musicXXX.bgw filename
8. Place in correct sound folder
